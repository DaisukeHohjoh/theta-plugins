## 説明
Automatic Face Blur BETA (自動顔ぼかし β版)は、THETAで静止画を撮影後、自動で画像を解析して顔にぼかしをかけます。ぼかしをかける前の写真とあわせて保存します。

## 新機能
RICOH THETA Xに対応しました。
